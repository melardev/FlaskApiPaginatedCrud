# Steps used to create this project
```bash
mkdir FlaskApiCrudPaginated
cd FlaskApiCrudPaginated
mkdir todos
mkdir shared
touch todos/models
# write Todo model
flask2 db init
flask2 db migrate -m 'initial'
flask2 db upgrade

# write the rest of your app

```
